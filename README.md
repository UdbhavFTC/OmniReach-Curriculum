# OmniReach-Curriculum
Overview

This repository contains a complete set of seven presentation decks designed to accompany a beginner-friendly FTC (FIRST Tech Challenge) programming course built using Android Studio and Java. Each presentation aligns with a corresponding video in the course and focuses on breaking down complex robotics concepts into clear, visual, and structured explanations. The material is designed not just to teach programming, but to make the learning process intuitive and approachable for students with little to no prior experience.

Purpose

The primary purpose of this repository is to provide a consistent and structured visual curriculum for FTC programming. These presentations act as a bridge between theoretical concepts and practical implementation, helping students understand not just how to code, but why certain approaches are used. In addition, the content serves as a foundational layer for accessibility-focused teaching approaches, where visual reinforcement plays a key role in improving comprehension.

Course Structure

The repository includes seven presentations that follow a logical progression through FTC programming. It begins with an introduction to the programming environment and overall structure of FTC systems, followed by the fundamentals of TeleOp coding and program logic. The course then moves into hardware components such as servos and motors, explaining both their physical behavior and their implementation in code. This is followed by drivetrain concepts, specifically the Mecanum drive, which introduces more advanced movement logic. Building on this, the course covers finite state machines as a method for structuring complex, multi-step actions within a robot. Finally, the series concludes with an introduction to autonomous programming, including commonly used tools and strategic considerations.

Design Approach

The presentations are designed with a strong emphasis on clarity and progression. Concepts are introduced gradually, with each topic building on the previous one to create a cohesive learning experience. Visual elements such as diagrams, flow representations, and simplified code structures are used to reinforce understanding and reduce cognitive overload. Rather than relying on dense text, the slides prioritize readability and explanation through guided instruction, making them suitable for both self-learning and classroom environments.

Application and Context

These materials were actively used in teaching sessions during the FTC season, including work with beginner students and in environments requiring more structured and visual instruction. They also formed the basis for further development into more advanced teaching systems, demonstrating their flexibility as both a standalone resource and a foundational curriculum. The presentations are intended to be adaptable, allowing educators and teams to modify them based on their specific needs, experience levels, and teaching styles.

Usage

This repository can be used as a companion to the video course, as a standalone teaching resource, or as a starting point for building customized FTC training programs. Educators can use the presentations to guide lessons, while students can use them for revision and reinforcement of key concepts. The structure of the content also makes it suitable for iterative improvement and integration into broader educational tools.

Closing Note

This repository represents an effort to make FTC programming more accessible, structured, and easier to approach for new learners. By combining clear explanations with visual support, it aims to reduce the initial barrier to entry and create a smoother learning experience for anyone getting started with robotics programming.
